Pseudo HTML

"Pseudo HTML" is not a recognized term within web development. The concept being referred to is likely CSS Pseudo-classes and Pseudo-elements, which are used to style specific states or parts of HTML elements without altering the HTML structure itself.

CSS Pseudo-classes target elements based on their state or interaction with the user. They are denoted by a single colon (:).

Examples include:-

  • :hover: Styles an element when the user's mouse cursor is over it.
  • :active: Styles an element when it is being activated (e.g., clicked).
  • :focus: Styles an element when it has received focus (e.g., from a keyboard tab).
  • :first-child: Styles an element if it is the first child of its parent.
  • :valid, :invalid: Styles form elements based on whether their input is valid or invalid.

CSS Pseudo-elements target specific parts of an element that are not distinct nodes in the HTML document tree. They are denoted by a double colon (::).

Examples include:-

  • ::before: Inserts content before the content of an element.
  • ::after: Inserts content after the content of an element.
  • ::first-line: Styles the first line of a block-level element
  • ::first-letter: Styles the first letter of a block-level element.
  • ::selection: Styles the portion of an element that is currently selected by the user.
